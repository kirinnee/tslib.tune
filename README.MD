# Tune

Audio Library, wrapper over Howlerjs, which gives for more robust functions such as easing

# Getting Started

Install via NPM 
```powershell
$ npm i @kirinnee/tune
```

or 

Install via Yarn
```powershell
$ yarn add @kirinnee/tune --prefer-offline
```

Using in browser

Attach scripts in `dist/` folder
```html
<script src="./dist/@kirinnee/tune.min.js"></script>
```
